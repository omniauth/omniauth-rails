# OmniAuthRails

**Ruby on Rails extensions to OmniAuth**

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'omniauth-rails'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omniauth-rails
